KaPlan Gottesdienste

WordPress Plugin für die Anzeige aktueller Gottesdienste und Termine

Installation

  1. Laden Sie das Plugin herunter und entpacken Sie es
  2. Laden Sie den Ordner kaplan_gottesdienste in das Verzeichnis /wp-content/plugins/ hoch
  3. Aktivieren Sie das Plugin im WordPress-Admin unter Plugins
  4. Konfigurieren Sie die Plugin-Einstellungen entsprechend Ihrer KaPlan-Installation

Shortcode-Verwendung

Fügen Sie den folgenden Shortcode in Ihre WordPress-Seiten oder Beiträge ein:

[ausgabe_kaplan server="ihre-server-domain.de" arbeitsgruppe="ihre-arbeitsgruppe" code="ihren-zugangscode"]
Wichtiger Hinweis: Verwenden Sie immer gerade Anführungszeichen ("") für die Parameter-Werte im Shortcode. Geschweifte Anführungszeichen ("") können zu Fehlern führen!

Template-Optionen

Das Plugin unterstützt drei verschiedene Darstellungsformate:

Template 1 (Standard)

Klassische Listen-Darstellung mit Datumsüberschriften:

Montag, 23. September 2025
10:00  HL. Messe (Pfarrkirche St. Marien, Pfr. Müller)
18:30  Abendgebet (Pfarrkirche St. Marien)

Shortcode für Template 1:

[ausgabe_kaplan server="ihre-server-domain.de" arbeitsgruppe="ihre-arbeitsgruppe" code="ihren-zugangscode" template="1"]

Template 2 (Tabellarisch)

Zweispaltige Tabellen-Darstellung mit Zeit- und Inhaltsspalten:

Montag, 23. September 2025
10:00 HL. Messe (Pfarrkirche St. Marien, Pfr. Müller)
18:30 Abendgebet (Pfarrkirche St. Marien)

Shortcode für Template 2:

[ausgabe_kaplan server="ihre-server-domain.de" arbeitsgruppe="ihre-arbeitsgruppe" code="ihren-zugangscode" template="2"]

Template 3 (Dreispaltig)

Dreispaltige Tabellen-Darstellung mit separaten Spalten für Zeit, Gottesdienst-Info und Ort:

Montag, 23. September 2025
10:00
HL. Messe
mit Orgelmusik
(Pfr. Müller)
 Pfarrkirche St. Marien
18:30
Abendgebet
(Diakon Weber)
 Kapelle
Neuerung in v1.8.6: In Template 2 und 3 wird die Leitungs-Information jetzt als letzte Zeile in der mittleren Spalte angezeigt, was für bessere Übersichtlichkeit sorgt.

Shortcode für Template 3:

[ausgabe_kaplan server="ihre-server-domain.de" arbeitsgruppe="ihre-arbeitsgruppe" code="ihren-zugangscode" template="3"]

Verfügbare Parameter

Parameter Beschreibung Standard
server Domain Ihres KaPlan-Servers erforderlich
arbeitsgruppe Ihre Arbeitsgruppe erforderlich
code Ihr Zugangscode erforderlich
template Darstellungsformat (1=Listen, 2=Zweispaltig, 3=Dreispaltig) 1
mode A=Kirchengruppiert, B=Chronologisch B
options Zusätzliche Optionen (E=Ereignis, U=Uhrzeit) -
leitung Leitungsformat (K/N/VN/TVN/etc.) -
days Anzahl Tage (ab heute) alle

Beispiel-Shortcodes

Einfache chronologische Anzeige:

[ausgabe_kaplan server="demo.kaplan-software.de" arbeitsgruppe="demo" code="demo123" mode="B"]

Mit Leitung und Uhrzeit:

[ausgabe_kaplan server="demo.kaplan-software.de" arbeitsgruppe="demo" code="demo123" leitung="TVN" options="U"]

Tabellarische Darstellung (2-spaltig):

[ausgabe_kaplan server="demo.kaplan-software.de" arbeitsgruppe="demo" code="demo123" template="2" mode="B"]

Dreispaltige Darstellung mit Leitung:

[ausgabe_kaplan server="demo.kaplan-software.de" arbeitsgruppe="demo" code="demo123" template="3" leitung="TN" mode="B"]
